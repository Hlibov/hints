Это самовызывающая функция предназначена для разчёта высоты экрана устройства.
Подключать файл full-screen-height.js необходимо в jquery.main.js пример здесь http://prntscr.com/1ny1y11
Значение высоты экрана устройства саписывается в css-переменную var(--full-viewport).
Данную css-переменную можно использовать как значение свойтва height пример здесь http://prntscr.com/1nxzinc
